# THE SILENT THREAD

**A narrative-driven, interactive horror game where memories unravel and the line between reality and illusion blurs.**

---

## Overview

*The Silent Thread* is an atmospheric, choice-based adventure game. Players explore eerie environments, solve puzzles, collect items, and make decisions that influence multiple endings. The story revolves around a mysterious thread guiding you through your past, confronting the Seamstress, and uncovering the truth behind your lost memories.

---

## Features

- Multiple chapters with branching narrative paths
- Investigation mechanics with interactive hotspots
- Puzzle challenges including keypads, circuits, and rune alignments
- Inventory system with collectible items influencing story outcomes
- Dice-based skill checks for stealth, evasion, and more
- Several unique endings:
  - **True Ending**
  - **Neutral Ending**
  - **Bad Ending**
  - **Secret Ending**
  - **Successor Ending**
